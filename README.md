 
VS Code Extensions Used
1. Auto Rename Tag

Automatically renames paired HTML/XML tags.

Example:
If you change:

<div>

to:

<section>

It will also update the closing tag automatically:

</section>

👉 Saves time and prevents mismatched tags.

2. Live Server  

Launches a local development server with live reload.

What it does:

Opens your project in the browser
Automatically refreshes when you save changes

👉 Makes development faster—no manual refresh needed.

3. Prettier – Code Formatter

Formats your code automatically to keep it clean and consistent.

What it does:

Fixes spacing
Aligns code
Keeps style consistent

👉 Helps maintain readable and professional code.

4. Google Fonts

Allows easy access to Google Fonts inside your project.

What it does:

Quickly search and insert fonts
Improves UI design

👉 Makes styling faster and more visually appealing.

5 Houston (Color Theme)

A Visual Studio Code theme that changes the editor’s color scheme.

What it does:

Improves code readability
Highlights syntax clearly (HTML, CSS, JavaScript)
Reduces eye strain during long coding sessions

👉 Helps you focus and makes your coding environment more comfortable.
